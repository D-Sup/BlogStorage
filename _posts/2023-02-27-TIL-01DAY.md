## overflow: hidden

```html
<div class="float-wrap">
  <p class="title-text">Ministry of Health</p>
  <p class="year-text">2015 - 2018</p>
</div>
```

```css
.title-text {
  font-size: 11px;
  font-weight: bold;
  color: #282828;
  float: left;
}

.year-text {
  font-size: 11px;
  font-weight: bold;
  color: #282828;
  float: right;
}

.float-wrap {
  overflow: hidden;
}
```

그 다음에 오는 HTML요소들이 float의 영향을 받지 않도록 함
